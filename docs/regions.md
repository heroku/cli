## `heroku regions`

list available regions for deployment

```
USAGE
  $ heroku regions [--common] [--json] [--private]

FLAGS
  --common   show regions for common runtime
  --json     output in json format
  --private  show regions for private spaces

DESCRIPTION
  list available regions for deployment
```
