## `heroku teams`

list the teams that you are a member of

```
USAGE
  $ heroku teams [--prompt] [--json]

FLAGS
  --json  output in json format

GLOBAL FLAGS
  --prompt  interactively prompt for command arguments and flags

DESCRIPTION
  list the teams that you are a member of

  Use heroku members:* to manage team members.
```
